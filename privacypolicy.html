Privacy Policy for Bynder Filter URL Generator (Chrome Extension)

Last updated: [06/07/2025]

1. Data Usage & Storage

  • This extension runs entirely in your browser.
  • No data is transmitted to any server or stored remotely.
  • All processing takes place locally within your browser.
  • User preferences (e.g. status filter settings) may be stored locally using Chrome’s storage API, but this data remains on your device.

2. Cookies & Tracking

  • This extension does not use cookies.
  • No tracking or analytics technologies are included.
  • Your browsing activity is not monitored or logged.

3. Third-Party Services

This extension does not use or integrate with any third-party services. It only interacts with your Bynder portal at *.bynder.com to generate search URLs.

4. Permissions

The extension requests limited Chrome permissions such as activeTab, tabs, webRequest, and access to *.bynder.com. These are required solely for reading search filters and generating URLs. No other browsing activity is accessed or used.

5. Security

The extension’s code is self-contained and does not execute or fetch code from any remote servers. All functionality is included in the packaged extension files.

6. Questions or Concerns

If you have any questions or concerns about this privacy policy, please contact the developer through the Chrome Web Store support form on the extension listing.
